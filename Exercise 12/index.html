JavaScript Condition Exercise

Your task is to modify the function numberChecker so that it returns true when the entered number is greater than or equal to 10, and false if it is less than 10.

Currently, the code checks if the number is equal to 6. You need to change this condition to check if the number is greater than or equal to 10.

Once you've made the changes, check the result in the Developer Console (Right-click and select "Inspect", then go to the "Console" tab).

Hint: You need to modify the condition inside the if statement.
