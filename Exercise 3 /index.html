JavaScript Math Exercise

Your task is to replace the strings in the code with the appropriate math expressions.

Rules:

  • Replace the strings to the right of the = with the correct math expression they describe.
  • Do not manually enter the answers to the equations. For example, writing const a = 9 would be incorrect, as 9 is the result of the expression.

Once you've replaced the strings with the correct expressions, check the results in the Developer Console (Right-click and select "Inspect", then go to the "Console" tab).

Hint: Remember to use standard JavaScript operators like +, *, and % for the math expressions.
